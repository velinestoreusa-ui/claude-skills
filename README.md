# Claude Skills — YouTube Production

Custom Claude Code skills for YouTube content production.

## Skills

| Skill | Trigger | Description |
|-------|---------|-------------|
| `llm-council` | `council this` | Run any decision through 5 AI advisors + final verdict |
| `humanizer` | `humanize this` | Remove AI writing patterns, make text sound natural |
| `save` | `/save` | Save session to LLM Wiki |

## Install

Copy any `.md` file (or folder) from `skills/` to `~/.claude/skills/`.

```bash
cp skills/llm-council.md ~/.claude/skills/
```
